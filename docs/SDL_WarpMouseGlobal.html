SDL_WarpMouseGlobal

画面上の指定された座標にマウスを移動させる

構文

int SDL_WarpMouseGlobal(int x, int y)

引数

xX座標
yY座標

戻り値

成功のとき0, 失敗のとき負の数のエラーコードを戻す. SDL_GetError()を呼んで詳細を知ることができる.

詳細

この関数はマウス移動イベントを発生させる.

一般的に, この関数に失敗するのは対応していない場合である.

バージョン

SDL 2.0.4以降
