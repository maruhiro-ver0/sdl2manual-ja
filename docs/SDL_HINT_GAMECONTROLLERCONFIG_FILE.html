SDL_HINT_GAMECONTROLLERCONFIG_FILE

追加のゲームコントローラー・データベース・ファイルを設定するヒント

詳細

ファイルの各行にはゲームコントローラー設定データが書き込まれている必要がある. SDL_gamecontroller.hを参照すること.

このヒントはSDL_Init(SDL_INIT_GAMECONTROLLER)の前に設定する必要がある. システムの初期化の後はSDL_GameControllerMappingForGUID()またはSDL_GameControllerAddMapping()で更新できる.
