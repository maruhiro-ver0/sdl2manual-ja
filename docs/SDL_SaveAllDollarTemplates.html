SDL_SaveAllDollarTemplates

現在の全てのジェスチャーのテンプレートを保存する

構文

int SDL_SaveAllDollarTemplates(SDL_RWops* dst)

引数

dst保存するSDL_RWops

戻り値

成功のとき保存したテンプレートの数, 失敗のとき0を戻す. SDL_GetError()で詳細を知ることができる.

バージョン

SDL 2.0.0以降
