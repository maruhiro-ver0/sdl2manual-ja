SDL_HapticStopAll

ハプティックデバイスで現在再生中の全ての効果を停止する

構文

int SDL_HapticStopAll(SDL_Haptic* haptic)

引数

haptic再生を停止するハプティックデバイス

戻り値

成功のとき0, 失敗のとき負の数のエラーコードを戻す. SDL_GetError()で詳細を知ることができる.

バージョン

SDL 2.0.0以降
