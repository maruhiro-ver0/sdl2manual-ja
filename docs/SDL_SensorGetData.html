SDL_SensorGetData

オープンされたセンサーの現在の状態を得る.

構文

int SDL_SensorGetData(SDL_Sensor *sensor, float *data, int num_values)

引数

sensor調査するSDL_Sensorオブジェクト
dataセンサーの現在の状態を代入するポインタ
num_valuesdataに書き込んだ値の数

戻り値

0, エラーの場合は-1を戻す.

詳細

値の数と意味はセンサーに依存する

バージョン

SDL 2.0.9以降
