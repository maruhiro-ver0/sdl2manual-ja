SDL_GDKRunApp

SDL GDKアプリケーションを初期化して起動する

構文

int SDL_GDKRunApp(SDL_main_func mainFunction, void *reserved)

引数

mainFunctionSDLアプリケーションのC言語スタイルのmain()であるSDL_main_func
reserved将来のための予約, 常にNULLにすること

戻り値

成功の0, エラーが発生したとき-1を戻す. SDL_GetError()で詳細を知ることができる.

バージョン

SDL 2.24.0以降
