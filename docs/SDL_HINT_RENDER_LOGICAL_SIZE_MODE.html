SDL_HINT_RENDER_LOGICAL_SIZE_MODE

SDL_RenderSetLogicalSize()の拡大縮小の方法を設定するヒント

値

"0"または"letterbox"上下/左右に黒帯を追加して画面にレンダリングする (デフォルト)
"1"または"overscan"端が画面外に描かれることを認めて画面全体にレンダリングする
