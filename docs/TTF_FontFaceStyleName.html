TTF_FontFaceStyleName

フォントのスタイル名を得る

構文

const char * TTF_FontFaceStyleName(const TTF_Font *font)

引数

font調査するフォント

戻り値

フォントのスタイル名を戻す.

詳細

この文字列はフォントファイルの内容によって決定する.

戻された文字列は内部に書き込まれているため呼び出し側で編集や解放を行ってはならない. この文字列は`font`をTTF_CloseFont()に渡すと他の部分と同様に無効になる.

バージョン

SDL_net 2.0.12以降
