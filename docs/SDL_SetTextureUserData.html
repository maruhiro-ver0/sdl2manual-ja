SDL_SetTextureUserData

ユーザ指定のポインタをテクスチャに関連付ける

構文

int SDL_SetTextureUserData(SDL_Texture* texture, void* userdata)

引数

texture更新するテクスチャ
userdataテクスチャに関連付けるポインタ

戻り値

成功のとき0, 失敗のとき-1を戻す.

バージョン

SDL 2.0.18以降
