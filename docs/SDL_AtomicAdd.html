SDL_AtomicAdd

アトミック変数に加算する

構文

int SDL_AtomicAdd(SDL_atomic_t* a, int v)

引数

a修正するSDL_atomic_tへのポインタ
v加算する値

戻り値

アトミック変数の加算前の値を戻す.

詳細

この関数もメモリバリアである.

注意: この関数の目的が分からないのであれば, 使うべきではない!

バージョン
