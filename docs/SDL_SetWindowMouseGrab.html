SDL_SetWindowMouseGrab

ウィンドウのマウスグラブモードを設定する

構文

void SDL_SetWindowMouseGrab(SDL_Window* window, SDL_bool grabbed)

引数

windowマウスグラブモードが設定される
grabbedSDL_TRUEのときマウスグラブする, SDL_FALSEのとき解除する

詳細

マウスをグラブするとカーソルの移動がウィンドウ内に制限される.

バージョン

SDL 2.0.16以降
