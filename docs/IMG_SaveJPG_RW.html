IMG_SaveJPG_RW

SDL_SurfaceをSDL_RWopsを経由してJPEG画像として保存する

構文

int IMG_SaveJPG_RW(SDL_Surface *surface, SDL_RWops *dst, int freedst, int quality)

引数

surface保存するSDL_Surface
dst画像の保存先のSDL_RWops
quality[0; 33]ならば低画質, [34; 66]ならば中, [67; 100]ならば高画質

戻り値

成功のとき0, エラーのとき-1を戻す.

バージョン

SDL_image 2.0.2以降
