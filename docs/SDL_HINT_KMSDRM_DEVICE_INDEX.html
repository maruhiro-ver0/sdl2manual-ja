SDL_HINT_KMSDRM_DEVICE_INDEX

どのKMSDRMデバイスを使用するかを設定するヒント

詳細

内部ではSDLはKMSDRMにアクセスするために"/dev/dri/cardNN"のデバイスをオープンする. "NN"にはデバイスの番号が入る.

SDLは最もよい番号(通常は0)を選ぼうとするが, アプリケーションやユーザはこのヒントに0から99の値を設定して強制的に選ばせることができる.

バージョン

SDL 2.24.0以降
