SDL_UnionFRect

2つの浮動小数点で表された長方形の両方を含む(最小の)長方形を求める

構文

void SDL_UnionFRect(const SDL_FRect *A, const SDL_FRect *B, SDL_FRect *result)

引数

A1番目のSDL_Rect
B2番目のSDL_Rect
resultAとBの両方を含むようなSDL_FRectを代入するポインタ

バージョン

SDL 2.0.22以降
