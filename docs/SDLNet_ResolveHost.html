SDLNet_ResolveHost

ホスト名とポートからネットワーク形式のIPアドレスへの解決を行う

構文

int SDLNet_ResolveHost(IPaddress *address, const char *host, Uint16 port)

引数

address解決されたIPアドレスとポートの代入先
host解決するホスト名 ("libsdl.org"の様な)
port接続先のポート

戻り値

成功のとき0, エラーのとき-1を戻す.

詳細

`host`がNULLの場合, 解決されたホストは`INADDR_ANY'となる.

ホストが解決されなかった場合, `address`にはINADDR_NONEが代入され, 関数は-1を戻す.

バージョン

SDL_net 2.0.0以降
