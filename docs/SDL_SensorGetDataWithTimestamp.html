SDL_SensorGetDataWithTimestamp

オープンされたセンサーの現在の状態と更新時間を得る.

構文

int SDL_SensorGetDataWithTimestamp(SDL_Sensor *sensor, Uint64 *timestamp, float *data, int num_values)

引数

sensor調査するSDL_Sensorオブジェクト
timestamp可能ならばセンサーが読み込まれたミリ秒の時間, 不可能なら0が代入されるポインタ
dataセンサーの現在の状態を代入するポインタ
num_valuesdataに書き込んだ値の数

戻り値

0, エラーの場合は-1を戻す.

詳細

値の数と意味はセンサーに依存する

バージョン

SDL 2.26.0以降
