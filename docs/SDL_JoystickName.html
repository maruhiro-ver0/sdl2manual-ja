SDL_JoystickName

ジョイスティックの実装から得た名前を得る

構文

const char* SDL_JoystickName(SDL_Joystick* joystick)

引数

joystick開いたジョイスティック

戻り値

選択したジョイスティックの実装から得た名前を戻す. 名前が見つからなければNULLを戻す. SDL_GetError()で詳細を知ることができる.

サンプルコード

/* 全ての接続されたジョイスティックの名前を表示する */
int num_joy, i;
num_joy = SDL_NumJoysticks();
printf("%d のジョイスティックを検知した¥n", num_joy);
for(i = 0; i < num_joy; i++)
{
    SDL_Joystick *joystick = SDL_JoystickOpen(i);
    printf("%s¥n", SDL_JoystickName(joystick));
}

バージョン

SDL 2.0.0以降
