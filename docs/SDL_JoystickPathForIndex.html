SDL_JoystickPathForIndex

ジョイスティックの実装に依存したパスを得る

構文

const char* SDL_JoystickPathForIndex(int device_index)

引数

device_index調査するジョイスティックの番号 (システムのN番目のジョイスティック)

戻り値

選択したジョイスティックのパスを戻す. パスがない場合はNULLを戻す. SDL_GetError()で詳細を知ることができる.

バージョン

SDL 2.24.0以降

関連項目(関数)

SDL_JoystickPath
SDL_JoystickOpen
