SDL_HINT_WINDOWS_DISABLE_THREAD_NAMING

SDLがMicrosoft Windowsのスレッドに名前を付けるか否かを指定するヒント

値

0スレッドに名前を付ける (デフォルト)
1スレッドに名前を付けない

デフォルト

デフォルトではSDLはMicrosoft Windowsのスレッドに名前を付ける.

詳細

このヒントが設定されていなければ, SDLはスレッドに名前付けするために例外0x406D1388を発生させる. これがSDLのデフォルトである.

このヒントが設定されていると, SDLはこの例外を発生させず, スレッドは無名となる. .Net言語の場合, デバッガで実行するときこれが必要となる.

バージョン

SDL 2.0.5以降
