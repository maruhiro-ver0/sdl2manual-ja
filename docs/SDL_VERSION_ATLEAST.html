SDL_VERSION_ATLEAST

コンパイル時のSDLのバージョンが指定のバージョンより新しいかチェックする

構文

SDL_VERSION_ATLEAST(X, Y, Z)

引数

Xメジャーバージョン
Yマイナーバージョン
Zパッチレベル(アップデートバージョン)

戻り値

コンパイル時のバージョン少なくともX.Y.Zならば真を戻す.

サンプルコード

if (! SDL_VERSION_ATLEAST(2,0,0)) {
    SDL_Log("SDL_VERSION %d は 2.0.0以前である", SDL_MAJOR_VERSION);
    return 1;
}
