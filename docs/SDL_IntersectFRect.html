SDL_IntersectFRect

浮動小数点で表された2つの長方形の交点を求める

構文

SDL_bool SDL_IntersectFRect(const SDL_FRect *A, const SDL_FRect *B, SDL_FRect * result)

引数

A1番目のSDL_FRect
B2番目のSDL_FRect
resultAとBの交点を代入するSDL_Rectのポインタ

戻り値

交点があるときSDL_TRUE, ないときSDL_FALSEを戻す.

詳細

resultがNULLのとき, この関数はSDL_FALSEを戻す.

バージョン

SDL 2.0.22以降
