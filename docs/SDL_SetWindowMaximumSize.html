SDL_SetWindowMaximumSize

ウィンドウのクライアント領域の最大サイズを設定する

構文

void SDL_SetWindowMaximumSize(SDL_Window* window, int max_w, int max_h)

引数

window設定するウィンドウ
max_wウィンドウの最大幅
max_hウィンドウの最大高さ

バージョン

SDL 2.0.0以降
