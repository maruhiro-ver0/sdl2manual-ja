SDL_GetNumTouchDevices

登録されたタッチデバイスの数を得る

構文

int SDL_GetNumTouchDevices(void)

戻り値

登録されたタッチデバイスの数を戻す

詳細

一部の環境では, SDLは実際に使われて初めてタッチデバイスを見つけることができる. よって, SDL_GetNumTouchDevices()は, デバイスが使えるにも関わらず0を戻すことがある. 全てのデバイスを少なくとも1回使えば, 値は正しくなる.

これはAndroid用のSDL2.0.1で修正された.

バージョン

SDL 2.0.0以降
