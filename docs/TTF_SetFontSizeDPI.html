TTF_SetFontSizeDPI

フォントのDPI単位で指定した解像度とサイズを変更する

構文

int TTF_SetFontSizeDPI(TTF_Font *font, int ptsize, unsigned int hdpi, unsigned int vdpi)

引数

fontサイズを変更するフォント
ptsize新しいポイントサイズ
hdpi水平方向DPI
vdpi垂直方向DPI

戻り値

成功のとき0, エラーのとき-1を戻す.

詳細

既に生成されたグリフがあればクリアされる.

バージョン

SDL_ttf 2.0.18以降
