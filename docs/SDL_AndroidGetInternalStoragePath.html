SDL_AndroidGetInternalStoragePath

アプリケーションが使う内部ストレージのパス名を得る

構文

const char* SDL_AndroidGetInternalStoragePath()

戻り値

成功のときアプリケーションが使う内部ストレージのパス名, 失敗のときNULLを戻す. SDL_GetError()で詳細を知ることができる.

詳細

パスはアプリケーション固有で, 他のアプリケーションは書き込めない.

典型的な内部ストレージのパス名は次のような形式である: /data/data/your.app.package/files

バージョン

SDL 2.0.0以降
