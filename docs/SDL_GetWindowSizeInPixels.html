SDL_GetWindowSizeInPixels

ウィンドウのサイズをピクセル数で得る

構文

void SDL_GetWindowSizeInPixels(SDL_Window *window, int *w, int *h)

引数

window幅と高さを得るウィンドウ
xウィンドウ幅のピクセル数を代入するポインタ. NULLでもよい
yウィンドウ高さのピクセル数を代入するポインタ. NULLでもよい

詳細

この関数の結果は高DPI, 例えば高DPI(Appleならば"Retina")に対応した環境でSDL_WINDOW_ALLOW_HIGHDPI付きで生成され, かつSDL_HINT_VIDEO_HIGHDPI_DISABLEDヒントで無効になっていないウィンドウの場合, SDL_GetWindowSize()の結果と恐らく異なる.

バージョン

SDL 2.26.0以降
