SDL_HasIntersection

2つの長方形に交点があるかチェックする

構文

SDL_bool SDL_HasIntersection(const SDL_Rect* A, const SDL_Rect* B)

引数

A1番目のSDL_Rect
B2番目のSDL_Rect

戻り値

交点があるときSDL_TRUE, ないときSDL_FALSEを戻す.

詳細

引数が両方ともNULLならば, この関数はSDL_FALSEを戻す.

バージョン

SDL 2.0.0以降
