SDL_AndroidSendMessage

SDLActivityにユーザコマンドを送信する

構文

int SDL_AndroidSendMessage(Uint32 command, int param)

引数

command0x8000以上のユーザコマンド
paramユーザパラメータ

詳細

メッセージをハンドルするためboolean onUnhandledMessage(Message msg)をオーバーライドする.

バージョン

2.0.22以降
