TTF_GlyphIsProvided

フォントの指定した16ビットコードポイントにグリフが提供されているかチェックする

構文

int TTF_GlyphIsProvided(TTF_Font *font, Uint16 ch)

引数

fontチェックするフォント
chチェックする文字コード

戻り値

指定の文字にグリフが提供されている場合は0以外, されていない場合は0を戻す.

詳細

このバージョンの関数は16ビット文字コードを引数とするため, 基本多言語面には対応しているが, 絵文字などの全てのユニコードを扱うには不十分である. 代わりに機能は同じだが32ビットコードポイントに対応したTTF_GlyphIsProvided32()を使うべきだ.

この関数を使う唯一の理由は, 最初のバージョンから存在しているから, ということのみである.

バージョン

SDL_ttf 2.0.12以降
