SDL_EncloseFPoints

複数の点の中を囲む最小の長方形の浮動小数点で表現された座標を求める

構文

SDL_bool SDL_EncloseFPoints(const SDL_FPoint *points, int count, const SDL_FRect *clip, SDL_FRect *result)

引数

points重なりを調べるSDL_FPointの配列
countpoints配列の数
clipクリップ領域のSDL_FPoint. NULLのとき全ての点を対象とする
result点を囲む最小のSDL_FPointを代入するポインタ

戻り値

長方形に囲まれる点があればSDL_TRUE, 点が全てクリップ領域の外ならばSDL_FALSEを戻す.

詳細

clipがNULLでなければ, クリップ領域の中の点のみ考慮される.

バージョン

SDL 2.0.22以降
