SDL_NumJoysticks

接続されたジョイスティックの数を得る

構文

int SDL_NumJoysticks(void)

戻り値

成功のとき接続されたジョイスティックの数, 失敗のとき負の数のエラーコードを戻す. SDL_GetError()で詳細を知ることができる.

バージョン

SDL 2.0.0以降
