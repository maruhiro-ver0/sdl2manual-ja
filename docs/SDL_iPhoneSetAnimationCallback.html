SDL_iPhoneSetAnimationCallback

Apple iOSのアニメーションコールバックを設定する

構文

int SDL_iPhoneSetAnimationCallback(SDL_Window * window, int interval, void (SDLCALL *callback)(void*), void *callbackParam)

引数

windowアニメーションコールバックを設定するウィンドウ
intervalcallbackが呼ばれるフレーム間隔
callback毎フレーム呼ばれる関数. 詳細を参照すること
callbackParamcallbackに渡されるポインタ

戻り値

成功のとき0, 失敗のとき負の数のエラーコードを戻す. SDL_GetError()で詳細を知ることができる.

詳細

callbackの関数プロトタイプは
void callback(void* callbackParam)

パラメータは

callbackParamSDL_iPhoneSetAnimationCallback()の引数として渡したcallbackParam

この関数はApple iOSのみ有効である.

詳細はREADME-ios.mdを参照すること.

この関数はSDL 2.0.4以前のSDL_iOSSetAnimationCallback()マクロを使ってアクセスすることもできる.

バージョン

SDL 2.0.0以降
