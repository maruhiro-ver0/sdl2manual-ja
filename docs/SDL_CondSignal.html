SDL_CondSignal

条件変数を待つ1つのスレッドを再開する

構文

int SDL_CondSignal(SDL_cond* cond)

引数

cond通知する条件変数

戻り値

成功のとき0, 失敗のとき負の数のエラーコードを戻す. SDL_GetError()で詳細を知ることができる.

バージョン

SDL 2.0.0以降

関連項目(関数)
