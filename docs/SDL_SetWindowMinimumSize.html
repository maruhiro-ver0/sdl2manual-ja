SDL_SetWindowMinimumSize

ウィンドウのクライアント領域の最小サイズを設定する

構文

void SDL_SetWindowMinimumSize(SDL_Window* window, int min_w, int min_h)

引数

window設定するウィンドウ
min_wウィンドウの最小幅
min_hウィンドウの最小高さ

バージョン

SDL 2.0.0以降
