SDL_net 2.0 API (アルファベット順)

関数

  1. SDLNet_AddSocket - ソケットセットにソケットを加える
  2. SDLNet_AllocPacket - UDPパケットを1つ確保する
  3. SDLNet_AllocPacketV - UDPパケットベクトルを確保する
  4. SDLNet_AllocSocketSet - SDLNet_CheckSockets()で使用するソケットセットを確保する
  5. SDLNet_CheckSockets - ソケットセットからデータを読み込めるかチェックする
  6. SDLNet_DelSocket - ソケットセットからソケットを取り除く
  7. SDLNet_FreePacket - UDPパケットを1つ解放する
  8. SDLNet_FreePacketV - UDPパケットベクトルを解放する
  9. SDLNet_FreeSocketSet - SDLNet_AllocSocketSet()で確保したソケットセットを解放する
  10. SDLNet_GetError - SDL_netの最後のエラーメッセージを得る
  11. SDLNet_GetLocalAddresses - このシステムのネットワークインターフェースのアドレスを得る
  12. SDLNet_Init - SDL_netを初期化する
  13. SDLNet_Linked_Version - プログラムにリンクされたSDL_netのバージョンを得る
  14. SDLNet_Quit - SDL_netを解放する
  15. SDLNet_ResizePacket - UDPパケットのペイロードを再確保する
  16. SDLNet_ResolveHost - ホスト名とポートからネットワーク形式のIPアドレスへの解決を行う
  17. SDLNet_ResolveIP - IPアドレスからホストの正式名の解決を行う
  18. SDLNet_SetError - SDLNet_GetErrorで戻されるエラーメッセージを設定する
  19. SDLNet_TCP_Accept - 指定のサーバソケットで接続を受け入れる
  20. SDLNet_TCP_AddSocket - ソケットセットにTCPソケットを加える
  21. SDLNet_TCP_Close - TCPネットワークソケットを閉じる
  22. SDLNet_TCP_DelSocket - ソケットセットからTCPソケットを取り除く
  23. SDLNet_TCP_GetPeerAddress - ソケットに関連付けられたリモートシステムのIPアドレスを得る
  24. SDLNet_TCP_Open - TCPソケットを開く
  25. SDLNet_TCP_OpenClient - クライアントTCPソケットを開く
  26. SDLNet_TCP_OpenServer - サーバTCPソケットを開く
  27. SDLNet_TCP_Recv - サーバでないソケットからデータを受信する
  28. SDLNet_TCP_Send - サーバでないソケットにデータを送信する
  29. SDLNet_UDP_AddSocket - ソケットセットにUDPソケットを加える
  30. SDLNet_UDP_Bind - アドレスをUDPソケットのチャネルにバインドする
  31. SDLNet_UDP_Close - UDPソケットを閉じる
  32. SDLNet_UDP_DelSocket - ソケットセットからUDPソケットを取り除く
  33. SDLNet_UDP_GetPeerAddress - ソケットとチャネルにバインドされたリモートシステムのIPアドレスを得る
  34. SDLNet_UDP_Open - UDPソケットを開く
  35. SDLNet_UDP_Recv - UDPソケットから1パケットを受信する
  36. SDLNet_UDP_RecvV - UDPソケットから複数のパケットを受信する
  37. SDLNet_UDP_Send - 指定のチャネルから1パケットを送信する
  38. SDLNet_UDP_SendV - 複数のパケットをパケットごとに指定されたチャネルに送信する
  39. SDLNet_UDP_SetPacketLoss - シミュレートする送信されたパケットの損失率を設定する
  40. SDLNet_UDP_Unbind - 指定のチャネルの全てのアドレスとのバインドを解除する

マクロ

  1. SDL_NET_COMPILEDVERSION - SDL_netの現在のバージョンナンバーのマクロ
  2. SDL_NET_VERSION_ATLEAST - コンパイル時のSDL_netのバージョンが少なくともX.Y.Zであるかチェックするマクロ
  3. SDLNet_SocketReady - ソケットセット内の指定したソケットからデータを読み込めるかチェックする
