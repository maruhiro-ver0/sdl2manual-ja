SDL_JoystickGetGUIDString

SDL_JoystickGUID構造体をASCII文字列に変換する

構文

void SDL_JoystickGetGUIDString(SDL_JoystickGUID guid, char* pszGUID, int cbGUID)

引数

guid文字列に変換するSDL_JoystickGUID
pszGUIDASCII文字列を書き込むバッファ
cbGUIDpszGUIDのサイズ

詳細

pszGUIDは少なくとも33byte必要である.

バージョン

SDL 2.0.0以降
