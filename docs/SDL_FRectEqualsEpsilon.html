SDL_FRectEqualsEpsilon

2つの長方形の座標, 幅, 高さが指定した誤差の範囲内であるかチェックする

構文

SDL_bool SDL_FRectEqualsEpsilon(const SDL_FRect *a, const SDL_FRect *b, const float epsilon)

引数

a長方形1へのポインタ
b長方形2へのポインタ
epsilon誤差

戻り値

範囲内ならば真, 範囲外ならば偽を戻す.

バージョン

2.0.22以降
