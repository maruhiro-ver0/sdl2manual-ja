SDL_MouseIsHaptic

マウスにハプティックの機能があるかチェックする

構文

int SDL_MouseIsHaptic(void)

戻り値

ハプティックのときSDL_TRUE, ハプティックでないときSDL_FALSEを戻す. SDL_GetError()で詳細を知ることができる.

バージョン

SDL 2.0.0以降
