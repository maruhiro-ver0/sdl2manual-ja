SDL_RWtell

SDL_RWopsデータストリームの現在の読込/書込位置を得る

構文

Sint64 SDL_RWtell(struct SDL_RWops* context)

引数

context現在のオフセット値を得るSDL_RWopsデータストリーム

戻り値

ストリームの現在のオフセット値を戻す. 失敗のとき-1を戻す.

サンプルコード

extern SDL_RWops *rw;
printf("ストリームの現在位置: %lld¥n", (long long) SDL_RWtell(rw));
if (SDL_RWseek(rw, 0, RW_SEEK_END) != -1) {
    printf("ストリームの最終位置: %lld¥n", (long long) SDL_RWtell(rw));
}

バージョン

SDL 2.0.10以降

詳細

SDL 2.0.10以前は, 実際にはSDL_RWopsのseekメソッドをRW_SEEK_CURから0byteで呼ぶマクロだった.
