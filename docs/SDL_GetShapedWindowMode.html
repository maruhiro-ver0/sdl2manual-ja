SDL_GetShapedWindowMode

非矩形ウィンドウの形のパラメータを得る

構文

int SDL_GetShapedWindowMode(SDL_Window* window, SDL_WindowShapeMode* shape_mode)

引数

windowパラメータを求める非矩形ウィンドウ
shape_modeパラメータが書き込まれる構造体. NULLのときウィンドウが非矩形かチェックされる

戻り値

ウィンドウが非矩形の場合, 0を戻し, shape_modeがNULLでなければデータが代入される. ウィンドウが非矩形でない場合, SDL_NONSHAPEABLE_WINDOWを戻す. 現在ウィンドウが形を失っている場合, SDL_WINDOW_LACKS_SHAPEを戻す.

バージョン

SDL 2.0.0以降
