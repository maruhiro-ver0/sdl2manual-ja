SDL_RenderLogicalToWindow

レンダラー上の倫理座標からウィンドウ上の実座標を得る

構文

void SDL_RenderLogicalToWindow(SDL_Renderer *renderer, float logicalX, float logicalY,int *windowX, int *windowY)

引数

rendererウィンドウ上の座標を得るレンダラ
logicalX論理X座標
logicalY論理Y座標
windowXウィンドウ上の実X座標の代入先
windowYウィンドウ上の実Y座標の代入先

詳細

レンダーが拡大縮小され論理レンダラーサイズが設定されている場合, 論理座標と実座標が異なる.

バージョン

SDL 2.0.18以降
