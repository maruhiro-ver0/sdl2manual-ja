SDL_GetNumVideoDisplays

利用可能なディスプレイの数を得る

構文

int SDL_GetNumVideoDisplays(void)

戻り値

成功のとき1以上の数を, 失敗のとき負の数のエラーコードを戻す. SDL_GetError()を呼んで詳細を知ることができる.

バージョン

SDL 2.0.0以降
