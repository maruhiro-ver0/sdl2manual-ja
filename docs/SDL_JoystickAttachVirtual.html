SDL_JoystickAttachVirtual

新しい仮想ジョイスティックを生成する

構文

int SDL_JoystickAttachVirtual(SDL_JoystickType type, int naxes, int nbuttons, int nhats)

戻り値

ジョイスティックのデバイス番号を戻す. 失敗のとき-1を戻す.

バージョン

SDL 2.0.14以降
