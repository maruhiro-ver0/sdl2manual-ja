SDL_RenderWindowToLogical

ウィンドウ上の実座標からレンダラ上の論理座標を求める

構文

void SDL_RenderWindowToLogical(SDL_Renderer * renderer, int windowX, int windowY, float *logicalX, float *logicalY)

引数

rendererレンダラ
windowXウィンドウ上の実X座標
windowYウィンドウ上の実Y座標
logicalX論理X座標の代入先を示すポインタ
logicalY論理Y座標の代入先を示すポインタ

詳細

レンダーが拡大縮小され, 論理レンダラサイズが設定されているとき論理座標と実座標が異なる.

バージョン

SDL 2.0.18以降
