SDL_RenderCopyF

テクスチャの一部をサブピクセル(副画素)単位で現在のレンダーターゲットにコピーする

構文

int SDL_RenderCopyF(SDL_Renderer* renderer, SDL_Texture* texture, const SDL_Rect* srcrect, const SDL_FRect* dstrect)

引数

rendererテクスチャの一部がコピーされるレンダラ
textureコピー元テクスチャ
srcrectコピー元の長方形へのポインタ. NULLのときテクスチャ全体
dstrectコピー先の長方形へのポインタ. NULLのときレンダーターゲット全体

バージョン

SDL 2.0.10以降

戻り値

成功のとき0, 失敗のとき負の数のエラーコードを戻す.
