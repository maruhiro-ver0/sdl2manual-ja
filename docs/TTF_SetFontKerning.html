TTF_SetFontKerning

フォントのカーニングの有効無効を設定する

構文

void TTF_SetFontKerning(TTF_Font *font, int allowed)

引数

fontカーニングを設定するフォント
allowedカーニングを有効にするとき0以外, 無効にするとき0

詳細

新たに開いたとき, フォントのカーニングは有効になっている. レンダリングの結果は良くなる場合が多いため, 特別な理由がない限り無効にしないのはよい方針である. (例えば一部のフォントはカーニングを無効にするとkerningがkemingのようになってしまう.)

バージョン

SDL_net 2.0.12以降
