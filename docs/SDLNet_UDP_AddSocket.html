SDLNet_UDP_AddSocket

ソケットセットにUDPソケットを加える

構文

SDL_FORCE_INLINE int SDLNet_UDP_AddSocket(SDLNet_SocketSet set, UDPsocket sock)

引数

setソケットを追加するソケットセット
socksetに追加するソケット

戻り値

セットに含まれるソケットの合計数(新たに加えたものを含む)を戻す. 既に満杯のとき-1を戻す.

詳細

これはUDPに特化したSDLNet_AddSocket()のラッパーである. この関数のドキュメントを参照すること.

バージョン

SDL_net 2.0.0以降
