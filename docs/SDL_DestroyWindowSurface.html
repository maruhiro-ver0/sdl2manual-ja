SDL_DestroyWindowSurface

ウィンドウに関連付けられたサーフェイスを破棄する

構文

int SDL_DestroyWindowSurface(SDL_Window *window)

引数

window変更するウィンドウ

戻り値

成功のとき0, エラーのとき負の数のエラーコードを戻す. SDL_GetError()を呼んで詳細を知ることができる.

バージョン

SDL 2.28.0以降
