SDL_LockSensors

センサーAPIへのマルチスレッドアクセスをロックする

構文

void SDL_LockSensors(void)

詳細

もし複数のスレッドでセンサーAPIの使用やイベント処理を行うならば, この関数でロックしてセンサーへのアクセスを保護する必要がある.

センサー一覧が変わらないこと保障されるため, API関数はセンサー番号を正しく受けることができる. また, センサーイベントは送られない.

バージョン

SDL 2.0.14以降
