SDL_RenderGetLogicalSize

レンダリングのデバイスに依存しない分解能を得る

構文

void SDL_RenderGetLogicalSize(SDL_Renderer *renderer, int *w, int *h)

引数

rendererレンダリングコンテキスト
w幅を代入するポインタ
h高さを代入するポインタ

詳細

メインレンダリングターゲットに使用する場合(例えばターゲットテクスチャが設定されていない): SDL_RendererにSDL_RenderSetLogicalSize()で論理サイズが設定されていなければwとhには0が代入される. そうでなければ論理幅と高さが代入される.

ターゲットテクスチャに使用する場合: 初めにwとhに0が代入されることはない. その後は設定された論理幅と高さが代入される.

バージョン

SDL 2.0.0以降
