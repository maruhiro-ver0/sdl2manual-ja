SDL_AudioStreamPut

変換/リサンプルしたいデータを音声ストリームに追加する

構文

int SDL_AudioStreamPut(SDL_AudioStream* stream, const void* buf, int len)

引数

streamデータを追加するストリーム
buf追加するデータへのポインタ
lenストリームに書き込むデータのバイト数

戻り値

成功のとき0, 失敗のとき-1を戻す.

バージョン

2.0.7以降
