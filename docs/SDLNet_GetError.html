SDLNet_GetError

SDL_netの最後のエラーメッセージを得る

構文

const char *SDLNet_GetError(void)

戻り値

UTF-8コードの最後のエラーメッセージを戻す.

詳細

エラーメッセージは, SDL_netのビルドによって, スレッドローカルデータであることもないこともありうる. エラーが発生しなかった場合でもエラーメッセージが設定されることがある. エラー文字列は公開APIが失敗を報告した直後のみ意味があり, それ以外が無視するべきである.

バージョン

SDL_net 2.0.0以降
