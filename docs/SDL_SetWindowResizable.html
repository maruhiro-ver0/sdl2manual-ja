SDL_SetWindowResizable

ウィンドウのサイズをユーザが変えられるかを設定する

構文

void SDL_SetWindowResizable(SDL_Window* window, SDL_bool resizable)

引数

windowサイズ変更可不可を変更するウィンドウ
resizableサイズ変更を許可するときSDL_TRUE, 許可しないときSDL_FALSE

詳細

この関数はウィンドウのSDL_WINDOW_RESIZABLEフラグの追加または取り除きを行い, ユーザによるウィンドウのサイズの変更を許可/不許可にする. もし既にウィンドウがその状態になっている場合は何も起こらない.

メモ: フルスクリーンウィンドウのときはサイズ変更可不可状態を変更できない.

バージョン

SDL 2.0.5以降
