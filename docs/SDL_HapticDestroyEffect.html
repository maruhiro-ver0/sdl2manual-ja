SDL_HapticDestroyEffect

ハプティックデバイスの効果を破棄する

構文

void SDL_HapticDestroyEffect(SDL_Haptic* haptic, int effect)

引数

haptic破棄する効果を持つハプティックデバイス
effect破棄する効果のID

詳細

効果が再生中ならば停止する. 効果はハプティックデバイスを閉じたとき自動的に破棄される.

バージョン

SDL 2.0.0以降
