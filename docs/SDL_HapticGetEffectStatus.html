SDL_HapticGetEffectStatus

指定のハプティックデバイスの現在の効果の状態を得る

構文

int SDL_HapticGetEffectStatus(SDL_Haptic* haptic, int effect)

引数

haptic状態を獲得するハプティックデバイス
effect状態を獲得する効果

戻り値

再生中でないとき0, 再生中のとき1, 失敗のとき負の数のエラーコードを戻す. SDL_GetError()で詳細を知ることができる.

詳細

デバイスがSDL_HAPTIC_STATUSに対応している必要がある.

バージョン

SDL 2.0.0以降
