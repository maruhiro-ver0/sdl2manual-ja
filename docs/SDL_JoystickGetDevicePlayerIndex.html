SDL_JoystickGetDevicePlayerIndex

ジョイスティックのプレイヤー番号を得る

構文

int SDL_JoystickGetDevicePlayerIndex(int device_index)

戻り値

ジョイスティックのプレイヤー番号を戻す. 有効でないとき-1を戻す.

詳細

この関数はジョイスティックをオープンする前に呼べる.

バージョン

SDL 2.0.9以降
