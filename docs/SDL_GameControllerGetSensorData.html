SDL_GameControllerGetSensorData

ゲームコントローラーのセンサーの状態を得る

構文

int SDL_GameControllerGetSensorData(SDL_GameController *gamecontroller, SDL_SensorType type, float *data, int num_values)

引数

gamecontroller状態を求めるSDL_GameController
typeセンサーの種類
dataセンサー状態の書込先ポインタ
num_values書き込んだ値の数

戻り値

成功のとき0, 失敗のとき-1を戻す.

詳細

dataに書き込まれる値の数と意味はセンサーによって異なる. 各センサー種類の詳細はSDL_sensor.hを見ること.

バージョン

SDL 2.0.14以降
