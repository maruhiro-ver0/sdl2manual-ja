SDL_WinRTRunApp

SDL/WinRTアプリケーションを初期化し起動する

構文

int SDL_WinRTRunApp(MainFunction mainFunction, void* reserved)

引数

mainFunctionSDLアプリのC言語スタイルのmain() 詳細を参照すること
reserved将来のための予約. NULLにすること

戻り値

成功のとき0, 失敗のとき-1を戻す. SDL_GetError()で詳細を知ることができる.

詳細

mainFunctionの関数プロトタイプは: 

int MainFunction(int argc, char **argv)

引数は:

argc引数の個数
argv引数の配列

バージョン

SDL 2.0.4以降
