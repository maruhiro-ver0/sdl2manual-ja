TTF_SetFontWrappedAlign

フォントの折り返し属性を設定する

構文

void TTF_SetFontWrappedAlign(TTF_Font *font, int align)

引数

font折り返し属性を設定するフォント
align新しい折り返し属性

戻り値

成功のとき0, エラーのとき-1を戻す.

詳細

折り返し属性は以下の内の1つである:

  • TTF_WRAPPED_ALIGN_LEFT (左に折り返す)
  • TTF_WRAPPED_ALIGN_CENTER (中央に折り返す)
  • TTF_WRAPPED_ALIGN_RIGHT (右に折り返す)

バージョン

SDL_ttf 2.20.0以降
