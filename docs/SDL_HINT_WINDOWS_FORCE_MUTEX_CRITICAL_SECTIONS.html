SDL_HINT_WINDOWS_FORCE_MUTEX_CRITICAL_SECTIONS

WindowsでのmutexでSDLに強制的にクリティカルセクションを使用させるか否かを設定するヒント

値

0可能ならばSRWロックを使用し, 不可能ならばクリティカルセクションを使用する (デフォルト)
1いかなる状況でもクリティカルセクションの使用を強制する

詳細

Windows 7以降では, Slim Reader/Writerロックが利用できる. これはパフォーマンスがより高く, カーネルリソースを消費せず, メモリの使用量がより少ない. SDLは古いバージョンのOSや, このヒントで強制されたときクリティカルセクションを使用する.
