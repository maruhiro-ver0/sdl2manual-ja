SDL_GameControllerGetAppleSFSymbolsNameForAxis

Apple環境のゲームコントローラーの指定の軸のsfSymbolsNameを得る

構文

const char *SDL_GameControllerGetAppleSFSymbolsNameForAxis(SDL_GameController *gamecontroller, SDL_GameControllerAxis axis)

引数

gamecontroller調査するゲームコントローラー
axisゲームコントローラーの軸

戻り値

sfSymbolsName, 名前が見つからなかったときNULLを戻す.

バージョン

SDL 2.0.18以降
