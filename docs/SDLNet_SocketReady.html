SDLNet_SocketReady

ソケットセット内の指定したソケットからデータを読み込めるかチェックする

構文

#define SDLNet_SocketReady(sock) _SDLNet_SocketReady((SDLNet_GenericSocket)(sock))

引数

sockチェックするソケット

戻り値

ソケットに新しいデータがある場合は0以外, それ以外の場合0を戻す.

詳細

After calling SDLNet_CheckSockets(), you can use this function on a socket that was in the socket set, to find out if data is available for reading. SDLNet_CheckSockets()を呼んだ後に, この関数をソケットセット内のソケットに対して使うことで, データを読み込めるかを調べることができる.

バージョン

SDL_net 2.0.0以降
