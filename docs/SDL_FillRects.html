SDL_FillRects

指定の色で塗りつぶした複数の長方形を高速に描く

構文

int SDL_FillRects(SDL_Surface* dst, const SDL_Rect* rects, int count, Uint32 color)

引数

dst描画の対象のSDL_Surface
rects塗りつぶされる領域のSDL_Rectの配列
count塗りつぶされる領域の配列の個数
color塗りつぶす色 (詳細を参照すること)

戻り値

成功のとき0, 失敗のとき負の数のエラーコードを戻す. SDL_GetError()を呼んで詳細を知ることができる.

詳細

colorは対象のサーフェイスのピクセル形式である必要がある. SDL_MapRGB()またはSDL_MapRGBA()で生成できる. 色にα値があっても, 単にそのα値が書き込まれ, ブレンドは行われない.

サーフェイスにクリップ領域が設定されていれば(SDL_SetClipRect()で設定できる), クリップ領域とrectにしたがって塗りつぶされる.

バージョン

SDL 2.0.0以降
