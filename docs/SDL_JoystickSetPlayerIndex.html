SDL_JoystickSetPlayerIndex

オープンされたジョイスティックにプレイヤー番号を設定する

構文

void SDL_JoystickSetPlayerIndex(SDL_Joystick* joystick, int player_index)

引数

joystickSDL_JoystickOpen()が戻したSDL_Joystick
player_indexこのコントローラーに設定するプレイヤー番号. -1のときプレイヤー番号をクリアしてプレイヤーLEDを滅灯する

バージョン

SDL 2.0.12以降
