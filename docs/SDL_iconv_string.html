SDL_iconv_string

文字列のコード間変換を行う

構文

char* SDL_iconv_string(const char* tocode, const char* fromcode, const char* inbuf, size_t inbytesleft)

戻り値

文字列を戻す. SDL_free()で解放する必要がある. 失敗のときNULLを戻す.

バージョン

2.0.0以降
