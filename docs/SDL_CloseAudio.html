SDL_CloseAudio

オーディオデバイスを閉じるための古い関数. 新しいプログラムは代わりにSDL_CloseAudioDevice()を使う方がよい

構文

void SDL_CloseAudio(void)

サンプルコード

extern SDL_AudioSpec want;
if (SDL_OpenAudio(&want, NULL) == 0) {
    SDL_PauseAudio(0);
    SDL_Delay(5000);  // オーディオコールバックを5秒間実行させる
    SDL_CloseAudio();
}

詳細

この関数は

SDL_CloseAudioDevice(1);

と同じである.

そして, 古いSDL_OpenAudio()関数を使った場合のみ使える.

バージョン

SDL 2.0.0以降
