SDL_size_mul_overflow

aとbの積を求めオーバーフローしたとき-1を戻す

構文

int SDL_size_mul_overflow (size_t a, size_t b, size_t *ret)

引数

a値a
b値b
ret積へのポインタ

戻り値

オーバーフローしたとき-1を戻す. オーバーフローしなかったときretに積を設定し0を戻す.

バージョン

SDL 2.24.0以降
