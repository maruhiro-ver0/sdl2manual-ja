TTF_SetDirection

書字方向のグローバルな設定を行う

非推奨

この関数はHarfBuzzのhb_direction_tの値をintにキャストしており, 全てのフォントにグローバルに影響を与えることを期待している. この関数の代わりにSDL_ttfの列挙体が使えてフォントごとに設定できるTTF_SetFontDirection()を使って欲しい.

これはグローバルな設定である. TTF_SetFontDirection()で設定された値が優先されるが, 設定がない場合はこの設定が使われる.

デフォルトはHB_DIRECTION_LTR(左から右)である.

構文

int TTF_SetDirection(int direction)

引数

directionhb_direction_tの値

戻り値

成功のとき0, SDL_ttfがHarfBuzzに対応していない場合は-1を戻す.

バージョン

SDL_ttf 2.0.18以降
