SDL_JoystickAttachVirtualEx

拡張されたプロパティを使用して仮想ジョイスティックを生成する

構文

int SDL_JoystickAttachVirtualEx(const SDL_VirtualJoystickDesc *desc)

引数

desc仮想ジョイスティックの定義(SDL_VirtualJoystickDesc)

戻り値

ジョイスティックのデバイス番号を戻す. エラーが発生したとき-1を戻す.

関連項目(構造体)

SDL_VirtualJoystickDesc

バージョン

SDL 2.24.0以降
