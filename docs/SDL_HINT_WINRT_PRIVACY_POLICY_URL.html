SDL_HINT_WINRT_PRIVACY_POLICY_URL

WinRTアプリのプライバシーポリシーのURLを指定するヒント

値

xWinRTアプリのプライバシーポリシーのURL

デフォルト

デフォルトではこのヒントは設定されない.

詳細

ネットワークが有効なWinRTアプリは全てユーザにプライバシーポリシーを閲覧できるようにしなければならない. Windows 8, 8.1, RTでは, Windows設定チャームでアプリからアクセスしてこのプライバシーポリシーを閲覧できることを義務付けている. SDLではアプリのプライバシーポリシーへのリンクのURLを追加するコードが提供されている.

アプリのプライバシーポリシーのURLを設定するためには, SDL_Init()関数を呼ぶ前にSDL_HINT_WINRT_PRIVACY_POLICY_URLを設定する. ヒントには正しいURLが含まれている必要がある. 例: "http://www.example.com"

デフォルトは""で, SDLは設定チャームにプライバシーポリシーへのリンクを追加しない. このヒントはアプリの初期化の途中のみで設定しなければならない.

アプリの「プライバシーポリシー」リンクのラベルの文字列は別のヒントSDL_HINT_WINRT_PRIVACY_POLICY_LABELで変更できる.

Windows Phoneでは, Microsoftはプライバシーポリシーのリンクを表示する標準のUIを提供しておらず, SDL_HINT_WINRT_PRIVACY_POLICY_URLは使えないので注意すること. ネットワークが有効なWindows Phoneアプリは別の方法でプライバシーポリシーを表示する必要がある.

バージョン

SDL 2.0.3以降
