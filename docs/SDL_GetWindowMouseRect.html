SDL_GetWindowMouseRect

ウィンドウのマウスの移動範囲を得る

構文

const SDL_Rect *SDL_GetWindowMouseRect(SDL_Window *window)

引数

windowウィンドウ

戻り値

ウィンドウのマウスの移動範囲へのポインタを戻す. 存在しない場合NULLを戻す.

バージョン

SDL 2.0.18以降
