SDL_IMAGE_VERSION

SDL_imageライブラリのコンパイル時のバージョンをバージョン構造体にセットするマクロ

構文

#define SDL_IMAGE_VERSION(X)                        \
{                                                   \
    (X)->major = SDL_IMAGE_MAJOR_VERSION;           \
    (X)->minor = SDL_IMAGE_MINOR_VERSION;           \
    (X)->patch = SDL_IMAGE_PATCHLEVEL;              \
}
