SDLNet_Linked_Version

プログラムにリンクされたSDL_netのバージョンを得る

構文

const SDLNet_version * SDLNet_Linked_Version(void)

戻り値

バージョン情報へのポインタを戻す.

詳細

この関数で得られるのは動的にリンクされたSDL_netのバージョンである. これとは異なり, SDL_NET_VERSION()マクロで得られるのはコンパイルした時点のSDL_netのヘッダのバージョンである.

戻されるのは静的な内部データであるため, 解放や修正を行ってはならない.

バージョン

SDL_net 2.0.0以降
