SDL_AudioStreamGet

変換/リサンプルされたデータを音声ストリームから得る

構文

int SDL_AudioStreamGet(SDL_AudioStream* stream, const void* buf, int len)

引数

stream変換されたデータを得るストリーム
buf音声データが書き込まれるバッファへのポインタ
len書き込まれるデータの最大バイト数

戻り値

ストリームから読み込んだバイト数を戻す, 失敗のとき-1を戻す.

バージョン

2.0.7以降
