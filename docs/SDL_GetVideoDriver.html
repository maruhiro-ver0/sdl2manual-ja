SDL_GetVideoDriver

組み込まれたビデオドライバの一覧から名前を得る

構文

const char* SDL_GetVideoDriver(int index)

引数

indexビデオドライバの番号

戻り値

与えられた番号のビデオドライバの名前を戻す

詳細

ビデオドライバの一覧は初期化中にチェックされた順番に並んでいる.

バージョン

SDL 2.0.0以降
