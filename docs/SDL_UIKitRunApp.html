SDL_UIKitRunApp

SDLアプリケーションを初期化して起動する

構文

int SDL_UIKitRunApp(int argc, char* argv[], SDL_main_func mainFunction)

引数

argcアプリケーションのmain()関数のargc引数
argvアプリケーションのmain()関数のargv引数
mainFunctionSDLアプリケーションのC言語スタイルのmain(), SDL_main_func

戻り値

mainFunctionが戻した値を戻す

バージョン

SDL 2.0.10以降
