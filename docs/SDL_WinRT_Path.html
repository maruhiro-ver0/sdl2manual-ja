SDL_WinRT_Path

WinRT / Windows Phoneパスの種類の列挙体

値

SDL_WINRT_PATH_INSTALLED_LOCATIONアプリのインストールされたルートディレクトリ. ここにあるファイルは読込専用となる
SDL_WINRT_PATH_LOCAL_FOLDERアプリのLocalディレクトリ. ファイルは書き込める
SDL_WINRT_PATH_ROAMING_FOLDERアプリのRoamingディレクトリ(Windows Phoneは対応していない). ここにあるファイルはネットワーク接続を経由して他のコンピュータにコピーできる
SDL_WINRT_PATH_TEMP_FOLDERアプリのTemporaryディレクトリ(Windows Phoneは対応していない). ファイルはいつでも削除できる

関連項目(関数)

SDL_WinRTGetFSPathUNICODE
SDL_WinRTGetFSPathUTF8

バージョン

SDL 2.0.3以降
