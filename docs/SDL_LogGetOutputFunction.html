SDL_LogGetOutputFunction

現在のログ出力関数を得る

構文

void SDL_LogGetOutputFunction(SDL_LogOutputFunction* callback, void** userdata)

引数

callback現在のログコールバック関数を代入するポインタ. (詳細を参照すること)
userdatacallbackに渡すポインタを代入するポインタ

詳細

callbackの関数プロトタイプは

void SDL_LogOutputFunction(void* userdata, int category, SDL_LogPriority priority, const char* message)

引数は

userdataSDL_LogSetOutputFunction()に渡されたuserdata
categoryメッセージのSDL_LOG_CATEGORY
propertyメッセージのSDL_LogPriority
message出力するメッセージ

バージョン

SDL 2.0.0以降
