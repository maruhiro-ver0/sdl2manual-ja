SDL_HapticSetGain

指定のハプティックデバイスのゲインを設定する

構文

int SDL_HapticSetGain(SDL_Haptic* haptic, int gain)

引数

hapticゲインを設定するハプティックデバイス
gain設定するゲイン. 0〜100の範囲

戻り値

成功のとき0, 失敗のとき負の数のエラーコードを戻す. SDL_GetError()で詳細を知ることができる.

詳細

デバイスがSDL_HAPTIC_GAINに対応している必要がある.

ユーザは環境変数SDL_HAPTIC_GAIN_MAXでゲインの最大値を0〜100で設定できる. SDL_HapticSetGain()はSDL_HAPTIC_GAIN_MAXを最大値とした値に線形に縮小される.

バージョン

SDL 2.0.0以降
