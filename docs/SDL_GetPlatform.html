SDL_GetPlatform

環境の名前を得る

構文

const char* SDL_GetPlatform(void)

戻り値

環境の名前を戻す. 正しい環境の名前が得られない場合は, "Unknown"で始まる文字列を戻す.

詳細

いくつかの対応した環境で戻される名前:
  • Windows
  • Mac OS X
  • Linux
  • iOS
  • Android

バージョン

SDL 2.0.0以降
