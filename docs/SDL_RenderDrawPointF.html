SDL_RenderDrawPointF

現在のレンダーターゲットに点をサブピクセル(副画素)単位で描く

構文

int SDL_RenderDrawPointF(SDL_Renderer* renderer, float x, float y)

引数

renderer点が描かれるレンダラー
x点のX座標
y点のY座標

バージョン

SDL 2.0.10以降

戻り値

成功のとき0, 失敗のとき負の数のエラーコードを戻す.
