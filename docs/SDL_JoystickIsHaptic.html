SDL_JoystickIsHaptic

ジョイスティックにハプティックの機能があるかチェックする

構文

int SDL_JoystickIsHaptic(SDL_Joystick* joystick)

引数

joystickハプティックの機能があるかチェックするジョイスティック

戻り値

ハプティックのときSDL_TRUE, ハプティックでないまたは失敗のときSDL_FALSEを戻す. SDL_GetError()で詳細を知ることができる.

バージョン

SDL 2.0.0以降
