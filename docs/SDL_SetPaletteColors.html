SDL_SetPaletteColors

パレットの色を設定する

構文

int SDL_SetPaletteColors(SDL_Palette* palette, const SDL_Color* colors, int firstcolor, int ncolors)

引数

palette設定するSDL_Palette
colorsパレットにコピーするSDL_Colorの配列
firstcolor設定するパレットの先頭の添え字
ncolors設定するパレットの数

戻り値

成功のとき0, 色を設定できなかったとき負の数のエラーコードを戻す. SDL_GetError()を呼んで詳細を知ることができる.

バージョン

SDL 2.0.0以降
