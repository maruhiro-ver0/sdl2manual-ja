SDL_HINT_MOUSE_RELATIVE_SYSTEM_SCALE

マウスが相対モードのときシステムのマウス加速曲線を使用するかを設定するヒント

値

0相対モードのとき加速しない (デフォルト)
1相対モードのときシステムのマウス加速曲線で加速する

明細

SDL_HINT_MOUSE_RELATIVE_SPEED_SCALEの設定はシステムの速度倍率を上書きする.
