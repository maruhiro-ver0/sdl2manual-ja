SDL_GetKeyName

キーコードから人が読める名前を得る

構文

const char* SDL_GetKeyName(SDL_Keycode key)

引数

key名前を得るSDL_Keycode

戻り値

UTF-8文字列へのポインタを戻す. 少なくとも次回この関数を呼ぶまでは有効である. もしこれ以上長い必要があれば, コピーしなければならない. キーコードの名前がなければ, この関数は空の文字列("")を戻す.

詳細

詳細はSDL_Keycodeを参照すること.

バージョン

SDL 2.0.0以降
