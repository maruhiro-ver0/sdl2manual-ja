SDL_RenderDrawLineF

現在のレンダーターゲットにサブピクセル(副画素)単位で直線を描く

構文

int SDL_RenderDrawLineF(SDL_Renderer* renderer, float x1, float y1, float x2, float y2)

引数

renderer直線が描かれるレンダラ
x1始点のX座標
y1始点のY座標
x2終点のX座標
y2終点のY座標

バージョン

SDL 2.0.10以降

戻り値

成功のとき0, 失敗のとき負の数のエラーコードを戻す.
