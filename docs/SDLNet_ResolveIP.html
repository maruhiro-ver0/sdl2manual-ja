SDLNet_ResolveIP

IPアドレスからホストの正式名の解決を行う

構文

const char *SDLNet_ResolveIP(const IPaddress *ip)

引数

ipホスト名に解決するIPアドレス

戻り値

ホスト名を戻す. 解決できないときNULLを戻す.

詳細

IPを解決できない場合, この関数はNULLを戻す. それ以外の場合, ホスト名が書き込まれた静的バッファへのポインタを戻す.

**注意**: この関数はスレッドセーフではない.

バージョン

SDL_net 2.0.0以降
