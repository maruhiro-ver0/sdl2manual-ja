SDL_GetCPUCacheLineSize

CPUのL1キャッシュラインのサイズを得る

構文

int SDL_GetCPUCacheLineSize(void)

戻り値

CPUのL1キャッシュラインのサイズをバイト単位で戻す.

詳細

これはマルチスレッド構造体のパッディングやSIMDプリフェッチのサイズを決めるときに有用である.

バージョン

SDL 2.0.0以降
