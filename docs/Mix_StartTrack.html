Mix_StartTrack

音楽オブジェクトのトラックの再生を開始する

構文

int Mix_StartTrack(Mix_Music *music, int track)

引数

music音楽オブジェクト
track再生するトラック番号. 0が最初のトラック

戻り値

成功のとき0, エラーまたは実装されていないとき-1を戻す.

詳細

この関数はGME形式に対してのみ使用できる.

バージョン

SDL_mixer 2.8.0以降
