SDL_HasIntersectionF

2つの浮動小数点で表された長方形に交点があるかチェックする

構文

SDL_bool SDL_HasIntersectionF(const SDL_FRect *A, const SDL_FRect *B)

引数

A1番目のSDL_FRect
B2番目のSDL_FRect

戻り値

交点があるときSDL_TRUE, ないときSDL_FALSEを戻す.

詳細

引数が両方ともNULLならば, この関数はSDL_FALSEを戻す.

バージョン

SDL 2.0.22以降
