エフェクト

これらの関数はエフェクト処理のためにある. エフェクトは効果音だけではない. エフェクトは全てSDL_mixerに組み込まれた, またはあなたが作成した後処理のルーチンである. エフェクトは個々のチャネル, または音楽を含む全てのチャネルを最終的にミックスしたストリームに対して適用できる.

登録

  1. Mix_RegisterEffect - エフェクト関数を登録する
  2. Mix_SetPostMix - 全てのミキシングが適用された後に呼ばれる関数を設定する
  3. Mix_UnregisterAllEffects - 全てのエフェクト関数を解除する
  4. Mix_UnregisterEffect - エフェクト関数を解除する
  5. Mix_EffectDone_t - チャネルの全てのループが終了し再生が完了したことを知らせるコールバック関数の型
  6. Mix_EffectFunc_t - エフェクトコールバック関数の型

組み込みエフェクト

  1. Mix_SetDistance - チャネルの距離を設定する
  2. Mix_SetPanning - チャネルのパンを設定する
  3. Mix_SetPosition - チャネルの位置を設定する
  4. Mix_SetReverseStereo - チャネルのステレオを逆にする
