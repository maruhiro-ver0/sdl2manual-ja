SDL_GameControllerPath

オープンしたゲームコントローラーの実装に依存したパスを得る

構文

const char *SDL_GameControllerPath(SDL_GameController *gamecontroller)

引数

gamecontrollerSDL_GameControllerOpen()が戻したゲームコントローラー識別子

戻り値

ゲームコントローラーの実装に依存したパスを戻す. パスがない場合, または識別子が不正な場合はNULLを戻す.

詳細

この関数はSDL_GameControllerNameForIndex()と同じパスを戻すが, (不安定な)デバイス順ではなくコントローラー識別子を引数にする.

バージョン

SDL 2.24.0以降
