SDL_RenderDrawRectF

現在のレンダーターゲットに長方形をサブピクセル(副画素)単位で描く

構文

int SDL_RenderDrawRectF(SDL_Renderer* renderer, const SDL_FRect* rect)

引数

renderer長方形が描かれるレンダラ
rect描かれる長方形へのポインタ. NULLのときレンダラの外枠全体

戻り値

成功のとき0, 失敗のとき負の数のエラーコードを戻す.

バージョン

SDL 2.0.10以降
