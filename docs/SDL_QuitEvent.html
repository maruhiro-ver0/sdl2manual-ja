SDL_QuitEvent

終了要求イベントの情報の構造体

フィールド

Uint32typeSDL_QUIT
Uint32timestampイベントのタイムスタンプ

サンプルコード

SDL_Event e;
for (;;) {
    SDL_PollEvent(&e);
    if (e.type == SDL_QUIT) {
        SDL_Log("プログラムは後 %i で終了する", e.quit.timestamp);
        break;
    }
}

詳細

SDL_QuitEventはSDL_Eventのメンバで, SDL_QUITが報告されたときに使われる. eventのquitフィールドからアクセスできる.

詳細はSDL_QUITを参照すること.
