TTF_SetFontStyle

フォントスタイルを設定する

構文

void TTF_SetFontStyle(TTF_Font *font, int style)

引数

fontスタイルを設定するフォント
style設定するスタイル. 論理和で複数設定が可能

詳細

新たにフォントスタイルを設定すると, 全てのグリフはキャッシュから削除される.

フォントスタイルは次の値の論理和である:

  • TTF_STYLE_NORMAL (通常)
  • TTF_STYLE_BOLD (太字)
  • TTF_STYLE_ITALIC (斜体)
  • TTF_STYLE_UNDERLINE (下線)
  • TTF_STYLE_STRIKETHROUGH (取り消し線)

バージョン

SDL_net 2.0.12以降
