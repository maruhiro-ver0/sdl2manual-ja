SDL_SoftStretch

高速で低品質な伸縮転送を2つの同一フォーマットのサーフェイス間で行う

構文

int SDL_SoftStretch(SDL_Surface* src, const SDL_Rect* srcrect, SDL_Surface* dst, const SDL_Rect* dstrect)

詳細

代わりにSDL_BlitScaled()を使用すること.

バージョン

バージョン2.0.0以降
