SDL_HINT_VIDEO_MAC_FULLSCREEN_SPACES

Mac OS XのSpacesのフルスクリーンの方針のヒント

値

0Spacesの対応を無効にする (FULLSCREEN_DESKTOPは無視され, SDL_WINDOW_RESIZABLEはフルスクリーンボタンをタイトルバーに表示しない)
1Spacesの対応を有効にする (FULLSCREEN_DESKTOPは無視されず, SDL_WINDOW_RESIZABLEはフルスクリーンボタンをタイトルバーに表示する) (デフォルト)

デフォルト

デフォルトではSpacesの対応は有効.

詳細

このヒントはMac OS Xのみに適用される.

SpacesはこのヒントにかかわらずOSがMac OS X Lion (10.7)より古ければ無効である. このヒントはウィンドウが生成される前に設定する必要がある.

バージョン

SDL 2.0.2以降
