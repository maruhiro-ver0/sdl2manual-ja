FAQ:iOS

何に対応しているか?

SDL 2.0.0はiOS 5.1.1以降に対応している. 旧バージョンのSDLはiOS 3.1.3以降に対応している.

最新の情報はインストールを参照すること.

開発

一般

_class_getMeta関数でEXC_BAD_ACCESSが発生する. どうすればよいか?

Xcodeメニューの Project → Edit Project Settings で, 下のLinkingセクションまでスクロールし, "Other Linker Flags" に -ObjC オプションを追加すること. もしターゲットにリンカの設定をしているならば, ターゲットを右クリックし, "Get Info"メニューオプションを選択して, このターゲットのレベルに加える必要がある.

iOS 3.1の場合, アプリケーションがローディングスクリーンで終了するのはなぜか?

おそらくこれらのデバイスはARMv7アーキテクチャではなくARMv6アーキテクチャを使っている. Xcodeメインプロジェクトウィンドウの "Active Architecture" をARMv6に変えて試してみること.
