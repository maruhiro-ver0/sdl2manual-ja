SDL_JoystickNumBalls

ジョイスティックのトラックボールの数を得る

構文

int SDL_JoystickNumBalls(SDL_Joystick* joystick)

引数

joystick開いたジョイスティック

戻り値

成功のときトラックボールの数, 失敗のとき負の数のエラーコードを戻す. SDL_GetError()で詳細を知ることができる.

バージョン

SDL 2.0.0以降
