SDL_GameControllerPathForIndex

オープンしたゲームコントローラーの実装に依存したパスを得る

構文

const char *SDL_GameControllerPathForIndex(int joystick_index)

引数

joystick_indexデバイスの0からSDL_NumJoysticks()までの番号

戻り値

ゲームコントローラーの実装に依存したパスを戻す. パスがない場合, または番号が不正な場合はNULLを戻す.

詳細

この関数はコントローラーを開く前に呼ぶことができる. joystick_indexはSDL_JoystickOpen()に渡すdevice_indexと同じである.

バージョン

SDL 2.24.0以降
