SDL_HINT_WINDOW_FRAME_USABLE_WHILE_CURSOR_HIDDEN

カーソルが見えないとき, ウィンドウのフレームやタイトルバーを操作できるか否かを指定するヒント

値

0カーソルが見えないときフレームを操作(動かす, サイズを変えるなど)できない
1カーソルが見えないときもウィンドウを操作できる (デフォルト)

デフォルト

デフォルトではカーソルが隠れていてもウィンドウのフレームを操作できる.

バージョン

SDL 2.0.4以降
