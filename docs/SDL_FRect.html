SDL_FRect

左上を基点とした長方形を定義する構造体

フィールド

floatx長方形の左上のX座標
floaty長方形の左上のY座標
floatw長方形の幅
floath長方形の高さ

サンプルコード

SDL_FRect rect;

rect.x = 0.0;
rect.y = 0.0;
rect.w = 32.0;
rect.h = 32.0;

詳細

SDL_FRectはスクリーン上の長方形の領域を定義する.
