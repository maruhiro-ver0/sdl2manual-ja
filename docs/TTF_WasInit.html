TTF_WasInit

SDL_ttfが初期化済みかチェックする

構文

int TTF_WasInit(void)

戻り値

初期化関数を呼び出した回数を戻す. TTF_Quit()の呼び出しはこの回数と一致していなければならない.

詳細

この関数は, ペアとなったTTF_Quit()による解放を除いたTTF_Init()の呼び出しでライブラリを初期化した回数を報告する.

簡潔に書くと: 0より大きい場合はライブラリは初期化済みで利用可能であり, 0の場合は初期化されていない.

戻り値の型は符号あり整数だが, 負の数を戻すことはない.

バージョン

SDL_net 2.0.12以降
