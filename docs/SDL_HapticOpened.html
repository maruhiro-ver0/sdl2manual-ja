SDL_HapticOpened

指定の番号のハプティックデバイスが開いているかチェックする

構文

int SDL_HapticOpened(int device_index)

引数

device_index開いているか調査する番号

戻り値

開いているとき1, 開いていないまたは失敗とき0を戻す. SDL_GetError()で詳細を知ることができる.

バージョン

SDL 2.0.0以降

関連項目(関数)
