SDL_UpperBlitScaled

サーフェイスを別のサーフェイスに拡大縮小してコピーする

構文

int SDL_UpperBlitScaled(SDL_Surface* src, const SDL_Rect* srcrect, SDL_Surface* dst, SDL_Rect* dstrect)

詳細

SDL_UpperBlitScaled()はSDL_BlitScaled()に置き換えられた. これは単なる分かりにくい名前のマクロである.

バージョン

SDL 2.0.0以降
