IMG_FreeAnimation

IMG_Animationを破棄して資源を解放する

構文

void IMG_FreeAnimation(IMG_Animation *anim)

引数

anim破棄するIMG_Animation

戻り値

新しいIMG_Animation, エラーのときNULLを戻す.

詳細

この関数を呼んだ後のanimポインタは有効ではない.

バージョン

SDL_image 2.6.0以降
