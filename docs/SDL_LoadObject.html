SDL_LoadObject

共有オブジェクトを動的に読み込み, オブジェクトハンドラへのポインタを戻す

構文

void* SDL_LoadObject(const char* sofile)

引数

sofile環境に依存したオブジェクトファイルの名前

戻り値

オブジェクトハンドラへのポインタを戻す. 失敗のときNULLを戻す. SDL_GetError()で詳細を知ることができる.

サンプルコード

#include "SDL.h"

/* 動的にmylib.soを読み込む */
SDL_LoadObject("mylib.so");

バージョン

SDL 2.0.0以降
