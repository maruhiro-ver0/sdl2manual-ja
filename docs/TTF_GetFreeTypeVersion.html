TTF_GetFreeTypeVersion

使用しているFreeTypeライブラリのバージョンを得る

構文

void TTF_GetFreeTypeVersion(int *major, int *minor, int *patch)

引数

majorメジャーバージョンの代入先へのポインタ. NULLも可
minorマイナーバージョンの代入先へのポインタ. NULLも可
patchパッチバージョンの代入先へのポインタ. NULLも可

詳細

この関数を呼ぶ前にTTF_Init()が呼ばれている必要がある.

バージョン

SDL_ttf 2.0.18以降
