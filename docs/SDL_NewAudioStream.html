SDL_NewAudioStream

新しい音声ストリームを生成する.

構文

SDL_AudioStream* SDL_NewAudioStream(SDL_AudioFormat src_format, Uint8 src_channels, int src_rate,SDL_AudioFormat dst_format, Uint8 dst_channels, int dst_rate)

引数

src_format変換前の音声形式
src_channels変換前のチャネル数
src_rate変換前の周波数
dst_format変換後の音声形式
dst_channels変換後のチャネル数
dst_rate変換後の周波数

戻り値

成功のとき0, 失敗のとき-1を戻す.

バージョン

2.0.7以降
