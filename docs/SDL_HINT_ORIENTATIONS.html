SDL_HINT_ORIENTATIONS

iOSで認めるデバイスの方向のヒント

値

LandscapeLeftデバイスの左が上
LandscapeRightデバイスの右が上
Portrait縦向き
PortraitUpsideDownデバイスの下が上

デフォルト

デフォルトでは全ての方向を認める.

サンプルコード

#include "SDL.h"

int main(int argc, char* argv[])
{
    SDL_SetHint(SDL_HINT_ORIENTATIONS, "LandscapeLeft LandscapeRight");
}

詳細

認めるUIの方向を限定する必要がある場合がある.

このヒントは, スペースで区切った上の一覧の方向に限定する.
