SDL_AudioCallback

オーディオデバイスがさらなるデータを必要としたとき呼ばれる関数

構文

typedef void (SDLCALL * SDL_AudioCallback) (void *userdata, Uint8 * stream, int len)

引数

userdataアプリケーション固有のSDL_AudioSpecのuserdataフィールドに保存したパラメータ
stream音声データバッファへのポインタ
lenバッファのバイト長
