SDL_LogOutputFunction

ログ出力関数のプロトタイプ型

構文

typedef void (SDLCALL *SDL_LogOutputFunction)(void *userdata, int category, SDL_LogPriority priority, const char *message)

引数

userdatauserdataとしてSDL_LogSetOutputFunction()に渡したデータ
categoryメッセージの分類(SDL_LogCategory)
priorityメッセージの重要度(SDL_LogPriority)
message出力するメッセージ

詳細

この関数はログ出力された新しいテキストがあればSDLから呼ばれる.
