SDL_GDKGetDefaultUser

GDKのデフォルトユーザハンドルへの参照を得る

構文

int SDL_GDKGetDefaultUser(XUserHandle *outUserHandle)

引数

outUserHandleデフォルトユーザハンドルを代入するポインタ

戻り値

成功のとき0, 失敗のとき-1を戻す.

詳細

これは事実上のXUserAddAsyncの同期版で, 常にデフォルトユーザが優先され, サインインUIが許可されます.

バージョン

SDL 2.28.0以降
