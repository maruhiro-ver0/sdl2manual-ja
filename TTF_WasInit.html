TTF_WasInit

int TTF_WasInit()

TrueTypeフォントAPIが初期化されたか調査する. 二重に初期化しないようにTTF_Initの前に呼んだり, TTF_Quitを呼ぶか決めるために使える.

戻り値: 既に初期化されているとき1, 初期化されていないとき0

サンプルコード

if(!TTF_WasInit() && TTF_Init()==-1) {
    printf("TTF_Init: %s\n", TTF_GetError());
    exit(1);
}
