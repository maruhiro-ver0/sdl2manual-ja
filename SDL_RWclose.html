SDL_RWclose

確保したSDL_RWopsを解放し閉じる

構文

int SDL_RWclose(struct SDL_RWops* context)

引数

context閉じるSDL_RWops

戻り値

成功のとき0, 失敗のとき負の数のエラーコードを戻す. SDL_GetError()で詳細を知ることができる.

サンプルコード

SDL_RWops *rw = SDL_RWFromFile("test.bin", "r");
if (rw != NULL) {
    extern Uint8 buf[256];
    SDL_RWread(rw, buf, sizeof (buf), 1);
    SDL_RWclose(rw);
}

詳細

ディスクへの書き込みに失敗した場合, この関数はエラーを報告するが, その場合でもSDL_RWopsは使えなくなるので注意すること. SDL_RWcloseは実際にはSDL_RWopsのcloseメソッドを呼ぶマクロである.
