SDL_SetError

SDLエラーメッセージを設定する

構文

int SDL_SetError(const char* fmt, ...)

引数

fmtprintf()形式の文字列形式
...fmt文字列の%トークンに一致する追加の引数

戻り値

常に-1を戻す.

サンプルコード

(1)

SDL_SetError("予期しない異常が発生した!");

(2)

int errorCode = 0;
...
errorCode = -37;
...
if (errorCode < 0)
    SDL_SetError("予期しない異常が発生した: エラーコード %d", errorCode);

詳細

この関数を呼ぶと, 以前のエラーメッセージは上書きされる.
