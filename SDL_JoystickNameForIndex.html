SDL_JoystickNameForIndex

ジョイスティックの実装から得た名前を得る

構文

const char* SDL_JoystickNameForIndex(int device_index)

引数

device_index調査するジョイスティックの番号 (システムのN番目のジョイスティック)

戻り値

選択したジョイスティックの実装から得た名前を戻す. 名前が見つからなければNULLを戻す. SDL_GetError()で詳細を知ることができる.

サンプルコード

/* 全ての接続されたジョイスティックの名前を表示する */
int num_joy, i;
num_joy = SDL_NumJoysticks();
printf("%d のジョイスティックを検知した\n", num_joy);
for (i = 0; i < num_joy; ++i) {
    printf("%s\n", SDL_JoystickNameForIndex(i));
}

詳細

この関数はジョイスティックを開く前に呼ぶことができる.
