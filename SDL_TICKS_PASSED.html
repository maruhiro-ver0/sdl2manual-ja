SDL_TICKS_PASSED

経過時間(ミリ秒)を比較する

構文

SDL_TICKS_PASSED(A, B)

引数

A最初の経過時間
B次の経過時間

戻り値

AがBを超えていれば真を戻す.

サンプルコード

/* 100ミリ秒待つならば, 次のようにすればよい: */
Uint32 timeout = SDL_GetTicks() + 100;
while (!SDL_TICKS_PASSED(SDL_GetTicks(), timeout)) {
    /* ... ここで時間が経過するまでの処理を行う */
}

バージョン

SDL 2.0.0以降
