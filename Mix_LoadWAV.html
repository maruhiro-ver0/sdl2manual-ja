Mix_LoadWAV

Mix_Chunk *Mix_LoadWAV(char *file)

file
読み込むサンプリング音声ファイル

サンプリング音声ファイルを読み込む. これは実際には Mix_LoadWAV_RW(SDL_RWFromFile(file, "rb"), 1) である. WAVE, AIFF, RIFF, OGG, VOCファイルを読み込むことができる.

メモ: この関数の前にMix_OpenAudioを呼ぶ必要がある. 出力の形式に合わせて音声が変換されるが, その変換は読み込み時に行われる.

戻り値: サンプリング音声のMix_Chunkへのポインタ. 失敗のときNULL

サンプルコード

// sample.wavをサンプリング音声として読み込む
Mix_Chunk *sample;
sample=Mix_LoadWAV("sample.wav");
if(!sample) {
    printf("Mix_LoadWAV: %s\n", Mix_GetError());
    // ここでエラーを処理する
}
