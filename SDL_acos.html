SDL_acos

xのアークコサインを得る

構文

double SDL_acos(double x)

引数

x浮動小数点のラジアンの値

戻り値

xのアークコサインを戻す

サンプルコード

    /* acos(x) = Pi/2 - asin(x) */
    SDL_Log("acos(0):\t\t%f", SDL_acos(0));
    SDL_Log("Pi/2 - asin(0):\t%f", M_PI / 2 - SDL_asin(0));

    /* acos(-x) = Pi  - acos(x) */
    SDL_Log("acos(-(-1)):\t%f", SDL_acos(-(-1)));
    SDL_Log("Pi - acos(-1):\t%f", M_PI - SDL_acos(-1));

詳細

y = acos(x) の定義は x = cos(y) である.
(-1 <= x <= 1)
(0 <= y <= PI)

バージョン

SDL 2.0.2以降
