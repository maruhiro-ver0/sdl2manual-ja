SDL_RenderFillRects

現在のレンダーターゲットに複数の長方形を描く

構文

int SDL_RenderFillRects(SDL_Renderer* renderer, const SDL_Rect* rects, int count)

引数

rendererレンダリングコンテキスト
rects描かれる塗りつぶした長方形のSDL_Rectの配列
countrectsの数

戻り値

成功のとき0, 失敗のとき負の数のエラーコードを戻す. SDL_GetError()で詳細を知ることができる.
